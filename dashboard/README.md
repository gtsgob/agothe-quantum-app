# Dashboard

This directory contains the dashboard files for the Agothe Quantum App, which are deployed to GitHub Pages via the `Dashboard-deploy.yml` workflow.

## Purpose

The dashboard provides a web interface for monitoring and interacting with the quantum application. Files in this directory are automatically deployed to GitHub Pages when changes are pushed to the main branch.

## Contents

- `Index.html` - Main dashboard HTML page
- `style.css` - Dashboard styling

## Deployment

The dashboard is automatically deployed to GitHub Pages at:
`https://gtsgob.github.io/agothe-quantum-app/dashboard/`

The deployment is handled by the `.github/workflows/Dashboard-deploy.yml` workflow.
